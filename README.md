# 个人作品集网站

这是一个使用 Next.js 和 Tailwind CSS 构建的个人作品集网站。

## 仓库信息

- 远程仓库地址：https://github.com/wwwqqqzzz/PersonalWebsite.git

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Radix UI

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 构建

```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
